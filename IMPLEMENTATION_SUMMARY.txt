BACKEND (6 Critical Issues Fixed)
────────────────────────────────────────────────────────────────────────────
[✓] BUG-001: Chat History Persistence
    Issue: Database schema mismatch (role/content vs user_message/assistant_message)
    Fix: Standardized to model fields; updated get_session_messages
    Impact: Chat history now saves and retrieves correctly

[✓] BUG-002: SECRET_KEY Security Hardcoding
    Issue: Predictable default key exposed in source code
    Fix: Added startup validator; requires .env override; rejects empty strings
    Impact: Prevents unauthorized JWT token forgery

[✓] BUG-003: Timezone Inconsistency
    Issue: Mixed utcnow() (deprecated) and timezone.utc calls
    Fix: Created _now() helper; replaced all 5 occurrences
    Impact: Python 3.12 compatible; consistent timezone-aware datetimes

[✓] BUG-004: Tracker Routes Auth Bypass
    Issue: get_current_user typed as Optional but not checked
    Fix: Replaced with require_user dependency on all routes
    Impact: Proper authentication enforcement; no AttributeError crashes

[✓] BUG-005: Admin Database Connection Leaks
    Issue: Manual SessionLocal() bypasses FastAPI DI
    Fix: All admin routes now use Depends(get_db)
    Impact: Proper connection pool management; no exhaustion under load

[✓] BUG-006: Admin Stats Column Inconsistency
    Issue: timestamp vs created_at mismatch across endpoints
    Fix: Standardized to created_at everywhere
    Impact: Admin dashboard stats now consistent
